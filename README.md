We are currently transitioning **TaskFlow** from a desktop-centric layout to a **mobile-first, gesture-driven** minimalist powerhouse.

### Phase 1: The "Invisible" UI
- [ ] **Bottom Navigation**: Moving all controls to a mobile-friendly bottom tab bar (Today / Add / All).
